## 🚀 Como executar localmente

1.  Clone este repositório:

    ``` bash
    https://github.com/dekomonte/react-jogo-da-velha
    cd jogo-da-velha-react
    ```

2.  Instale as dependências:

    ``` bash
    npm install
    ```

3.  Execute a aplicação:

    ``` bash
    npm run start dev
    ```

4.  Abra no navegador:

        http://localhost:3000
